7.1.1 #how_to.mk_utils.cfg In the config

The main method of creating utilities - in config. Use map $utils. As keys, specify the names of utilities or groups.

@use 'mlut' with (
	$utils: (
		'D': ('', 'n'),
		'xl_P': 10e,
	)
);

// CSS
.D {
	display: block;
}

.D-n {
	display: none;
}

.P10e {
	padding: 10em;
}

@media (min-width: 1200px) {
	.xl_P10e {
		padding: 10em;
	}
}
