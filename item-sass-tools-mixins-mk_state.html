7.2.5 #sass-tools.mixins.mk_state mlu-mk-state

Generates values for utilities in various states. Names and values of states are set in the settings. Usually this mixin is not used separately because it is called inside mlu-mk-util.

@include mlu-mk-state(".Bgc-mn0", ("h")) {
	background-color: #60f;
}

CSS

.Bgc-mn0 {
	background-color: #60f;
}

.h_Bgc-mn0:hover {
	background-color: #60f;
}
Parameters:
  • $selector
    selector of CSS rule. Can be string or &.
    Defaults to: &
  • $states
    list of state names to be generated.
    Defaults to: map-keys($mlu-util-states)
  • $no-state
    number flag. If equal to 1, then a value without state will be generated as in the example above.
    Defaults to: 1
