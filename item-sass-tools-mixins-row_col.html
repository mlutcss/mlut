7.2.10 #sass-tools.mixins.row_col mlu-row-col

Generate styles for row__col element.

.item {
	@include mlu-row-col;
}
Parameters:
  • $display
    display type. Can be value of CSS display property.
    Defaults to: inline-block
  • $gutters
    map with column gutters values on breakpoints. You can also pass a number if you need the same gutter on all breakpoints. Custom gutters only work when the $css-vars = false.
    Defaults to: $mlu-grid-gutters
  • $calc
    bool flag for generation CSS calc() fallback. It can needed only for "col" $type.
    Defaults to: false
  • $type
    service parameter which used only by mixins-wrappers. It needed for reusable mlu-row-col for making other row elements, because the most of styles for them is the same.
    Defaults to: "col"
  • $css-vars
    use CSS custom properties for gutters. Works as a similar flag in the settings.
    Defaults to: $MLU-FLAG-CSS-VARS
