7.3.6 #sass-tools.settings.utils Utilities

Settings affecting the values and generation of the core utilities.

Parameters:
  • $mlu-Maw-fx0
    fixed width for cards. This is value for Maw-fx0 utility.
    Defaults to: 288px
  • $mlu-My-range
    count of Mt and Mb utilities.
    Defaults to: 4
  • $mlu-P-range
    count of P utilities.
    Defaults to: 4
  • $mlu--Gg-range
    number of -Gg utility values in su.
    Defaults to: 5
  • $mlu-Z-range
    z-index range. Counted from zero.
    Defaults to: 4
  • $mlu-Z-max
    z-index max value.
    Defaults to: 9999
  • $mlu-Ord-range
    order range. Counted from -1.
    Defaults to: 4
  • $mlu-Ff-ss0
    sans serif font stack contains default fonts on different OS.
  • $mlu-Ff-m0
    monospace font stack contains default fonts on different OS.
  • $mlu-Trsdu-df0
    default transition duration.
    Defaults to: 0.3s
  • $mlu-util-states-psd-cls
    map with abbreviations of pseudo-classes for utilities.
  • $mlu-util-states-psd-elm
    map with pseudo-elements abbreviations for utilities.
  • $mlu-util-states
    map with abbreviations of utility state names. Maps of pseudo-classes and pseudo-elements merge into it.
