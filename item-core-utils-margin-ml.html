4.8.2 #core-utils.margin.ml Ml

Toggle example guides Toggle HTML markup

Margin left. You can set left margin in the grid units(gg, c) or space units(su). So you can set the offset for layout elements.

  • Ml-a - Responsive.
  • Ml0 - Responsive.
  • Ml-2gg..Ml2gg - the range includes 2 negative margins and 2 positive.
  • Ml1gg - it is needed to return the default column gutter. Responsive.
  • Ml1c..Ml10c - available margin in c on all breakpoints, starting with md.
Example

Ml3c_md Ml0_lg Ml-a_xl

  • 3 of 12 columns on `md`
  • 0 on `lg`
  • auto on `xl`

Ml2gg

2 grid-gutters

Ml4c_md Ml1gg_xl

  • 4 of 12 columns on `md`
  • 1 grid-gutter on `xl`
