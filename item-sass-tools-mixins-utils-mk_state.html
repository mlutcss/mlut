8.2.2.1 #sass-tools.mixins.utils.mk_state mk-state

Generates values for utilities in various states. Names and values of states are set in the settings. Usually this mixin is not used separately because it is called inside mk-util.

@include mk-state('.Bgc-mn0', ('h')) {
	background-color: #60f;
}

CSS

.Bgc-mn0 {
	background-color: #60f;
}

.h_Bgc-mn0:hover {
	background-color: #60f;
}

$post $pre = null $no-state = 1 - number flag. If equal to 1, then a value without state will be generated as in the example above. $selector = & - selector of CSS rule. Can be string or &. $this-util = null $custom-sel = null
