7.2.12 #sass-tools.mixins.util_wrappers mk-util wrappers

Mixins-wrappers over mlu-mk-util and mk-util-directions, in which the names of utilities and CSS properties are written. Only a map with values is required. All have the same parameters.

The names of all mixins begin with mlu-mk- and end with the name of the utility they create: mlu-mk-Maw, mlu-mk-Fw etc. The exception is mlu-mk-Xy, which creates positioning utilities. 2 hyphens are written in the names of mixins for custom utilities: mlu-mk--Gcc. Mixins for groups of utilities with directions are marked separately.

Mixins for common utilities:

  • Positioning: Z
  • Box-model: W, Maw, Mih
  • Typografy: Fw, Fz, Ff, C, Lh
  • Visual: Bgc, Bdc, Bdw, Bdrs, Trf, Op

For custom utilities:

  • -Gcc

For utils with directions:

  • Xy
  • M
  • P
  • Bdxy

Example:

$Bdc-map: (
	"sc2": #333,
	"gr0": #eee,
);

@include mlu-mk-Bdc($Bdc-map);

CSS

.Bdc-sc2 {
	border-color: #333;
}

.Bdc-gr0 {
	border-color: #eee;
}
Parameters:
  • $map
    map with utilities names values and CSS properties values.
  • $bp
    breakpoint on which utility values will be generated.
    Defaults to: null
  • $states
    list of state names.
    Defaults to: null
