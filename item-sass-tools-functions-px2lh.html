7.1.15 #sass-tools.functions.px2lh mlu-px2lh

Wrapper on mlu-px2em for converting to line-height units. Works similarly.

mlu-px2lh(24px, 14px) => 1.71
mlu-px2lh(24px) => 1.5
Parameters:
  • $number
    number in px that will be converted.
  • $divider
    parent font size in px.
    Defaults to: $mlu-base-font-size
