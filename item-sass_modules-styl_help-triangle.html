8.3.8 #sass_modules.styl_help.triangle msh-triangle

Generate blank for border triangle.

.block {
	@include msh-triangle(14px);
}

CSS

.block {
	width: 0px;
	height: 0px;
	border: 14px transparent solid;
}
Parameters:
  • $size
    triangle size. For create triangle you need to add border-color for one of side after this mixin. For example: border-top-color: #f00.
