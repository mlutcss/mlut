7.3.4 #sass-tools.settings.general General

Settings that affect the work of the mixins, functions and all toolkit.

Parameters:
  • $MLU-BROWSER-FONT-SIZE
    font size for converting px to rem. 16px used because is default font size in settings of most browsers.
    Defaults to: 16px
  • $mlu-su-cf
    value of space unit. Use this factor to change su and for calculations.
    Defaults to: 4
  • $mlu-su
    space unit in px. Depends on $mlu-su-cf. Use this variable to set the values in px.
    Defaults to: 4px
  • $mlu-base-font-size
    base font size on website. Is used for rems fallback in row elements and in mlu-px2em default font size.
    Defaults to: $MLU-BROWSER-FONT-SIZE
  • $mlu-grid-default-gtr
    default gutter on large width screen. Used as fallback for --mlu-gg.
    Defaults to: 16px
  • $mlu-main-directions
    Map with directions for css properties and values generation. The keys are the first letters of the directions, and the values are directions.
    Example: ("t": top).
  • $mlu-grid-gutter-fallback
    Fallback for grid gutter's width if CSS calc() not supported
    Defaults to: 1.5%
