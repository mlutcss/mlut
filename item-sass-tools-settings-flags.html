7.3.3 #sass-tools.settings.flags Flags

Flags for core compilation.

Parameters:
  • $MLU-FLAG-CALC-FALLBACK
    turn on CSS calc() fallback for grid utils on md+ breakpoints. Type: bool.
    Defaults to: false
  • $MLU-FLAG-COL-OFFSET
    turn on generation grid columns offset utils. Type: bool.
    Defaults to: true
  • $MLU-FLAG-CSS-VARS
    turn on usage of CSS variables. Allowable values:
    false - turn off
    1 - turn on with fallback
    2 - turn on. With this value, most fallbacks will be disabled for browsers that don't support custom properties. Be careful.
    Defaults to: 1
  • $MLU-FLAG-FLEX-PE
    turn on utilities with progressive enhancement for flexbox.
    Defaults to: false
