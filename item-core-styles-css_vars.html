1.1 #core-styles.css_vars CSS variables

Global CSS custom properties which used in mlut.

Parameters:
  • --ml-gg
    grid gutter.
    Defaults to: 0.75rem
  • --ml-gscc
    number of columns in the grid system. Sets by $gs-columns. Available only with $legacy-grid-system flag enabled
    Defaults to: 12
