8.1.2.3.1 #sass-tools.functions.common.utils.convert_util_value convert-util-value

Determines the type of utility and, depending on type, calls the converter from the $util-value-converters map. If the utility type is not recognized, the default converter is called. If the $utils-value-converters does not contain a converter for this type, the utility value is returned without conversion. Else the result of the converter work is returned.

@debug ml.convert-util-value('2r', 'Fns'); // 2rem
@debug ml.convert-util-value(100, 'Fnw'); // 100
Parameters:
  • $util-val
    utility value
  • $name
    utility name
  • $section
    section in the registry in which utility was defined
    Defaults to: 'utils'
