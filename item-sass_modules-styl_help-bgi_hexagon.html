8.3.1 #sass_modules.styl_help.bgi_hexagon msh-bgi-hexagon

Generate hexagon with CSS linear-gradient() in background-image. See demo here.

.block {
	@include msh-bgi-hexagon($bgc: (#306, #d15));
}

CSS

.block {
	box-sizing: border-box;
	background-repeat: no-repeat;
	background-position: 0px 0px, 100% 0px, 50% 50%, 0px 102%, 100% 102%;
	background-size: 51% 27%, 51% 27%, 100% 48%, 51% 28%, 51% 28%;
	background-image: linear-gradient(150deg, transparent 50%, #306 50%), linear-gradient(-150deg, transparent 50%, #306 50%), linear-gradient(#306, #d15), linear-gradient(-330deg, transparent 50%, #d15 50%), linear-gradient(330deg, transparent 50%, #d15 50%);
}
Parameters:
  • $edge-ang
    linear-gradient angle of hexagon edges. Can be only number in deg.
    Defaults to: 150deg
  • $edge-cs
    is color-stop in linear-gradient of hexagon edges. Can be number with unit.
    Defaults to: 50%
  • $bgc
    hexagon color. It can be single color or list in linear-gradient value format. For example: (167deg, blue 50%, #f0f, #000). Angle in gradient can be only number in deg. If you pass gradient value, top hexagon edge get first color, bottom edge get last color and hexagon body get full gradient.
    Defaults to: #000
  • $body-size
    background-size of linear-gradient of hexagon body.
    Defaults to: (100% 48%)
  • $change
    bool flag for generate a change in color or size of an existing hexagon.
    Defaults to: false
