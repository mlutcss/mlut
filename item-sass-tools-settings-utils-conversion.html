8.3.5.1 #sass-tools.settings.utils.conversion Conversion to CSS

Settings of conversion utilitie values to CSS values. All variables below are maps.

Parameters:
  • $util-value-types
    value types and lists of utilities that belong to this type.
  • $util-unit-aliases
    utility unit aliases and corresponding CSS units.
  • $util-custom-units
    custom unit names and their number values.
  • $util-css-keywords
    keyword aliases and corresponding CSS global keywords.
